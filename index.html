Welcome to Brute OS

Brute OS is a Linux-based operating system designed for Offensive and Defensive security
Currently it is in alpha stage, and is continously being developed and contributed by Arijit Bhowmick [sys41x4]

Features

Powerful command-line interface

Brute OS includes a powerful command-line interface that allows you to easily perform complex security-related tasks.

Wide range of security tools pre-installed

Brute OS comes with a wide range of pre-installed security tools, so you don't have to waste time installing them yourself.

Customizable interface

Brute OS allows you to customize the interface to suit your needs, making it easy to use and navigate.

[A] Add brute-os-experimental Repository to APT Sources

Currently brute-os-experimental branch is accessable to use before a beta version of this OS is pushed

Add brute-os-experimental branch in your repository as : 

sudo sh -c 'echo "deb [arch=amd64] http://brute-os.github.io/deb/brute-os brute-os-experimental main contrib non-free non-free-firmware" > /etc/apt/sources.list.d/deb.brute-os.list'

Available Brute OS Key Servers:

1. keys.openpgp.org
2. keyserver.ubuntu.com

Add Brute OS PGP Key as:

>> Using keys.openpgp.org 
gpg --keyserver keys.openpgp.org --recv-key D9ABF9BF2C52565D
gpg --export --armor D9ABF9BF2C52565D | sudo apt-key add -
>> Using keyserver.ubuntu.com 
gpg --keyserver keyserver.ubuntu.com --recv-key D9ABF9BF2C52565D
gpg --export --armor D9ABF9BF2C52565D | sudo apt-key add -
